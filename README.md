# BathTunes
A game

Quick tutorial from the game of the game so you know what you are looking at in the code:
THIS IS A GAME WHERE YOU ROLEPLAY AS A DUCK THAT EITHER COLLECTS OR QUACKS AWAY OBJECTS USING VARIOUS STYLES OF INPUT SUCH AS MOUCE AND KEYBOARD. 
TRY TO DEFLECT AS FEW BUBBLES AS POSSIBLE AS THEY WILL GRANT YOU POINTS IF THEY COLIDE WITH YOU. QUACK EVERYTHING ELSE AWAY OR JUST DODGE THOSE
OBJECTS IF YOU CAN AS THEY ARE ENEMIES WHO WILL INSTEAD INSTANT KILL YOU IF THEY DO LIKEWISE. AT LEVEL 3 A NEW ENEMY WILL BEGIN TO APPEAR - THE
SQURTER. IT WILL FIRE UNBLOCKABLE JETS OF WATER AT YOUR POSITION SO BE SURE TO RAPIDLY NAVIGATE FROM SIDE TO SIDE TO DODGE THOSE DEADLY ATACKS. 
FINALLY - AT LEVEL 5 - YOU WILL ENCOUNTER THE BOSS ENEMY. IT IS COMPLETELY UNBLOCKABLE SO PRIORITISE DEFLECTING OTHER ENEMIES TO GET AROUND IT.
 
I would appreciate anyone who sees this to playtest for bugs and try to improve the source code accordingly.
